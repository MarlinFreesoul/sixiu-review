## 知识点整理格式说明

每章知识点采用统一格式：
```
章
├── 节
│   ├── 一、要点标题
│   │   ├── 要点内容（核心概念、定义、特征）
│   │   ├── 举例分析说明（结合典型案例）
│   │   └── 材料分析题/论述题举例
│   ├── 二、...
```

## 题型备考策略

| 题型 | 分值特征 | 备考重点 |
|------|---------|---------|
| 单选题 | 每题1分 | 记忆知识点细节，区分易混淆概念 |
| 多选题 | 多选/漏选/错选均不得分 | 记忆知识点下的多个要点（如四个伟大、五个原则） |
| 判断题 | 每题1分 | 辨析概念正误，注意绝对化表述 |
| 简答题 | 每题5-8分 | 罗列要点即可，不需展开 |
| 论述题 | 每题10-15分 | 知识点+解释+举例+总结 |
| 材料分析题 | 每题10-15分 | 阅读材料→找对应知识点→用原理解释→结合材料 |
