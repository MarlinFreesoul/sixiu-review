## 题型说明

| 题型 | 特征 | 来源参考 |
|------|------|---------|
| 单选题 | 知识点直接考察，四选一 | 教师作业详情 + 分章节练习题 |
| 多选题 | 知识点综合，多选/漏选/错选均不得分 | 教师作业详情 + 分章节练习题 |
| 判断题 | 概念辨析，正误判断 | 教师作业详情 + 分章节练习题 |
| 简答题 | 直接问知识点，需要罗列要点 | 教材课后习题 + 25道大题 |
| 论述题 | 需要展开论述，结合知识点+实际 | 教材各章思考题 |
| 材料分析题 | 给一段材料，用知识点分析 | 本质同论述题，需引用原理+结合材料 |

## 答题策略

- **选择题/判断题**：记牢知识点细节，注意概念区分
- **简答题**：列出要点即可，不需要过度展开
- **论述题/材料分析题**：知识点+解释+结合材料/举例+总结
