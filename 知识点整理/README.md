# 知识点整理

## 核心文件

**👉 [知识点树状总览.md](./知识点树状总览.md)** — 单一文件，完整知识树

格式：`章 → 节 → 一/二/三要点 → 要点内容 → 举例分析说明 → 材料分析题/论述题举例`

覆盖：绪论 + 第一章~第六章，每节末标注对应作业题型（单选/多选/判断）及来源。

## 答题策略

- **选择题/判断题**：记牢树状总览中的知识点细节。多选题特别关注"X个大点"（如三个意味着、四个伟大、五个原则等）——多选题经常考
- **简答题**：列出要点即可
- **论述题/材料分析题**：知识点 + 解释 + 结合材料/举例 + 总结（树状总览中每节附有例题）
