# 第五章 遵守道德规范 锤炼道德品格

## 第一节 社会主义道德的核心与原则

### 一、坚持马克思主义道德观

**要点内容：**

**（一）道德的起源与本质**

| 维度 | 内容 |
|------|------|
| **起源** | 劳动是道德起源的**首要前提**；社会关系是道德赖以产生的**客观条件**；人的自我意识是道德产生的**主观条件** |
| **本质** | 道德属于**上层建筑**范畴，是一种**特殊的社会意识形态**；是反映社会经济关系的特殊意识形态；是社会利益关系的特殊调节方式；是一种**实践精神** |

**（二）道德的功能与作用**
- 三大基本功能：**认识功能、规范功能、调节功能**
- 认识功能：反映社会关系特别是社会经济关系的功效与能力
- 规范功能：在正确善恶观引导下规范社会成员行为
- 调节功能：通过评价等方式指导和纠正人们的行为和实践活动
- 作用：巩固特定社会的经济基础和上层建筑；提高人的精神境界、促进人的自我完善、推动人的全面发展

**（三）社会主义道德的先进性**
- 是社会主义经济基础的反映
- 是对人类优秀道德资源的批判继承和创新发展
- 克服了以往阶级社会道德的片面性和局限性，坚持**以为人民服务为核心，以集体主义为原则**

---

### 二、坚持以为人民服务为核心

**要点内容：**
- 为人民服务是坚持**历史唯物主义**的必然要求
- 是中国共产党践行的**根本宗旨**
- 是社会主义道德观的**集中体现**
- 是社会主义经济基础和人际关系的客观要求
- 是**先进性要求和广泛性要求**的统一
- 是社会主义道德**区别于和优越于**其他社会形态道德的**显著标志**

**材料分析/论述题举例：**
> 为什么说"为人民服务"是社会主义道德的核心？结合现实谈谈你的理解。

---

### 三、坚持以集体主义为原则

**要点内容：**

**（一）集体主义是调节社会利益关系的基本原则**
1. 强调**国家利益、社会整体利益和个人利益的辩证统一**
2. 强调**国家利益、社会整体利益高于个人利益**
3. 重视和保障**个人的正当利益**

**（二）集体主义的三个层次**

| 层次 | 要求 |
|------|------|
| 最高层次 | 无私奉献、一心为公（对共产党员和先进分子的要求） |
| 中间层次 | 先公后私、先人后己（对先进分子的要求） |
| 基本层次 | 顾全大局、遵纪守法、热爱祖国、诚实劳动（对公民最基本的道德要求） |

**举例分析说明：**
- 抗疫期间：医护人员牺牲个人安全奔赴一线（先公后私），普通公民自觉居家隔离（遵纪守法、顾全大局）

**材料分析/论述题：**
> 结合集体主义的三个层次，谈谈大学生应如何正确处理个人利益与集体利益的关系。

## 第二节 吸收借鉴优秀道德成果

### 一、传承中华传统美德

**要点内容：**

**（一）中华传统美德的基本精神**

| 精神 | 说明 |
|------|------|
| 重视整体利益，强调责任奉献 | "先天下之忧而忧，后天下之乐而乐" |
| 推崇仁爱原则，注重以和为贵 | "己所不欲，勿施于人" |
| 注重人伦关系，重视道德义务 | 五伦关系（父子、君臣、夫妇、长幼、朋友） |
| 追求精神境界，向往理想人格 | "富贵不能淫，贫贱不能移，威武不能屈" |
| 强调道德修养，注重道德践履 | "吾日三省吾身" |

**（二）反对两种错误思潮**
1. **复古论**：认为道德建设的最终目标就是恢复中国"固有文化"
2. **虚无论**：认为中国传统文化在今天已失去价值和意义

---

### 二、发扬中国革命道德

**要点内容：**

**（一）定义：** 中国革命道德是中国共产党人、人民军队、一切先进分子和人民群众在中国革命、建设、改革中所形成的优秀道德

**（二）主要内容：**
1. 为实现社会主义和共产主义的理想信念而奋斗
2. 全心全意为人民服务
3. 始终把革命利益放在首位
4. 树立社会新风，建立新型人际关系
5. 修身自律，保持节操

**（三）当代价值：**
- 有利于加强和巩固社会主义和共产主义的理想信念
- 有利于培育和践行社会主义核心价值观
- 有利于引导人们树立正确的道德观
- 有利于培育良好的社会道德风尚

**举例分析说明：**
- 雷锋精神、焦裕禄精神、张思德精神等都是中国革命道德的具体体现

---

### 三、借鉴人类文明优秀道德成果

**要点内容：**

- 坚持马克思主义立场、观点、方法
- 坚持**以我为主，为我所用**

## 第三节 投身崇德向善的道德实践

### 一、遵守社会公德

**要点内容：**

**（一）公共生活的特征：** 活动范围的广泛性、活动内容的开放性、交往对象的复杂性、活动方式的多样性

**（二）社会公德的内容：**

| 规范 | 说明 |
|------|------|
| **文明礼貌** | 调整和规范人际关系的行为准则 |
| **助人为乐** | 把帮助他人视为自己应做之事 |
| **爱护公物** | 对社会共同劳动成果的珍惜和爱护 |
| **保护环境** | 尊重自然、顺应自然、保护自然 |
| **遵纪守法** | 全体公民都必须遵循的基本行为准则 |

**（三）网络生活中的道德要求：**
- 正确使用网络工具
- 加强网络文明自律
- 营造良好网络道德环境

---

### 二、恪守职业道德

**要点内容：**

**（一）职业道德的内容：**

| 规范 | 说明 |
|------|------|
| **爱岗敬业** | 干一行、爱一行、钻一行，精益求精、尽职尽责 |
| **诚实守信** | 诚实劳动、合法经营、信守承诺、讲求信誉 |
| **办事公道** | 公平、公正，不损公肥私，不以权谋私 |
| **热情服务** | 在本职岗位上为群众服务 |
| **奉献社会** | 兢兢业业为社会和他人做贡献 |

**（二）树立正确的择业观和创业观：**
- 树立崇高的职业理想
- 服从社会发展的需要
- 做好充分的择业准备
- 培养创业的勇气和能力

**举例分析说明：**
- 大国工匠：精益求精、爱岗敬业
- "劳动最光荣、劳动最崇高、劳动最伟大、劳动最美丽"

---

### 三、弘扬家庭美德

**要点内容：**

**（一）注重家庭、家教、家风**
- 家庭和睦则社会安定，家庭幸福则社会祥和，家庭文明则社会文明

**（二）家庭美德的内容：**
- **尊老爱幼**：子女孝敬赡养父母长辈，父母抚育爱护子女
- **男女平等**：夫妻权利和义务平等、人格地位平等
- **夫妻和睦**：互敬互爱、互助互让
- **勤俭持家**：勤劳致富、量入为出
- **邻里互助**：相互尊重对方的人格、民族习惯、生活方式

**（三）树立正确的恋爱观：**
- 不能把友谊当爱情；不能错置爱情的地位
- 不能片面或功利化地对待恋爱
- 不能只重过程不顾后果
- 不能因失恋而迷失人生方向

**材料分析/论述题：**
> 结合家庭美德的内容，谈谈家风建设对社会道德建设的重要意义。

---

### 四、锤炼个人品德

**要点内容：**

**（一）涵养高尚道德品格**
- 形成正确的道德认知和道德判断
- 激发正向的道德认同和道德情感
- 强化坚定的道德意志和道德信念

**（二）道德修养重在践行**
- 掌握道德修养的正确方法
- 向道德模范学习
- 参与志愿者服务活动

**（三）积极引领社会风尚**
- **知荣辱**：荣辱观对个人思想行为具有动力、导向和调节作用
- **讲正气**：坚持真理、坚持原则、坚持同歪风邪气作斗争
- **作奉献**：奉献精神是社会责任感的集中体现
- **促和谐**：促进自我身心和谐、个人与他人和谐、个人与社会和谐、人与自然和谐

**材料分析/论述题举例：**
> 结合"知荣辱、讲正气、作奉献、促和谐"的要求，谈谈大学生如何锤炼个人品德。
